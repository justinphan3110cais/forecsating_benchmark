### create an .env file here with:
```
OPENAI_API_KEY="your_openai_api_key"
SERPER_API_KEY="your_serper_api_key"
```

## Install dependencies:
```
pip3 install -r ../requirements.txt
```

### Then run manifold with
```[python]

python3 manifold.py --model gpt-4o --input_file data/manifold_03012024_small.json --prompt_type og --breadth 7

```
